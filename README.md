# Sky Weather App 🌤️

A modern, responsive, and beautiful weather application built with Kotlin and Jetpack Compose. Sky keeps you updated with current weather conditions, accurate hourly and daily forecasts, and lets you manage your favorite locations easily.

## Features ✨

*   **Current Location Weather:** Instantly view the weather conditions for your current area using GPS.
*   **Hourly & Daily Forecasts:** Stay prepared with detailed hourly graphical forecasts and a 7-day outlook.
*   **Favorite Locations:** Save multiple cities using an interactive Google Maps interface to check weather globally.
*   **Dynamic Localization:** Fully supports both English and Arabic with localized UI components.
*   **Unit Preferences:** Switch seamlessly between Metric, Imperial, and Standard units.
*   **Offline Mode Support:** View the latest cached weather data even when you lose your internet connection. 
*   **Smooth Animations & UI:** Clean, polished Jetpack Compose components focusing on a neat "Edge-to-Edge" experience.

## Tech Stack 🛠️

*   **Language:** Kotlin
*   **UI Toolkit:** Jetpack Compose (Material 3)
*   **Architecture:** MVVM (Model-View-ViewModel) + Clean Architecture principles
*   **Dependency Injection:** Koin
*   **Networking:** Retrofit + OkHttp
*   **Local Database:** Room Database
*   **Coroutines & Flow:** For asynchronous tasks and reactive data streams
*   **Maps:** Google Maps Compose
